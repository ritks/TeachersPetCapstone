Place your Manrope `.ttf` files in this folder:

- `Manrope-VariableFont_wght.ttf` (recommended, variable font), or
- Static files:
  - `Manrope-Regular.ttf`
  - `Manrope-Medium.ttf`
  - `Manrope-SemiBold.ttf`
  - `Manrope-Bold.ttf`

This app loads fonts from `/fonts/...` via `src/theme/tokens.css`.
